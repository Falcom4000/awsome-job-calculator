# 工作资产自动评分程序

一个纯前端本地计算的工作评分工具。用户填写工作信息后，程序把这份工作视为一项人力资本资产，从当前收益、稳定性、持有友好度、职业成长、流动性和个人匹配度六个维度计算总分、评级、雷达图和建议。

## Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- lucide-react
- html2canvas
- html-to-image
- qrcode
- @vercel/analytics
- @notionhq/client

## Product Notes

- 所有计算在浏览器本地完成，不上传用户输入。
- 支持简略模式和详细模式；两种模式输出同一套评分结果。
- 总分采用六维加权：当前收益 25%、稳定性 15%、持有友好度 20%、职业成长 20%、流动性 15%、个人匹配度 5%。
- 当前数据层已经建立 schema，并填入第一批公开来源数据；评分逻辑仍在逐步接入数据层。
- `requires.md` 是本地需求参考文件，不要求纳入 git。
