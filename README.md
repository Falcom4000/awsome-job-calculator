# Job Calculator

Starter scaffold for a new calculator project.

## Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- lucide-react
- html2canvas
- html-to-image
- qrcode
- @vercel/analytics
- @notionhq/client

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000 after the dev server starts.
